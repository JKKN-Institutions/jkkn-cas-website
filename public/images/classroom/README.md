# Classroom Images

This directory requires three classroom images for the Digital Classroom page:

## Required Images

1. **classroom-1.jpg** - Student in blue uniform
   - Shows student using digital learning technology
   - Recommended dimensions: 1200x900px (4:3 aspect ratio)
   - Format: JPG, optimized for web

2. **classroom-2.jpg** - Student in yellow uniform
   - Shows student engaged in digital classroom learning
   - Recommended dimensions: 1200x900px (4:3 aspect ratio)
   - Format: JPG, optimized for web

3. **classroom-3.jpg** - Students in classroom
   - Shows students in classroom setting with modern learning facilities
   - Recommended dimensions: 1200x900px (4:3 aspect ratio)
   - Format: JPG, optimized for web

## Image Optimization

- Use JPG format for photographs
- Optimize file size for web (aim for < 200KB per image)
- Maintain 4:3 aspect ratio for consistent display
- Ensure images are high quality but web-optimized

## Usage

These images are displayed in a responsive 3-column grid on the Digital Classroom page:
- Desktop/Tablet: 3 images side by side
- Mobile: 3 images stacked vertically

The images use Next.js Image component for automatic optimization and lazy loading.
